# AI Customer Feedback Analyzer

A beginner-friendly AI project that analyzes customer reviews and predicts whether the feedback is positive or negative.

Built using Python, Streamlit, Hugging Face Transformers, Pandas, and GitHub Copilot Free.

## Features

- Analyze one customer review at a time
- Predict positive or negative sentiment
- Display a confidence score
- Upload a CSV file containing multiple reviews
- Analyze all reviews in the CSV file
- Show total, positive, and negative review counts
- Download the analyzed CSV results

## Tech Stack

- Python
- Streamlit
- Hugging Face Transformers
- PyTorch
- Pandas
- GitHub Copilot Free
